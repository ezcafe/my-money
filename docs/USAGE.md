## Getting Started

### First Steps

1. **Login**: Use your passkey to log in via Pocket ID authentication
2. **Create Your First Account**: Navigate to **Accounts** from the main menu and create an account (e.g., "Checking Account", "Savings Account")
3. **Start Tracking**: Use the calculator on the home page to quickly add transactions

### Understanding the Interface

- **Home Page**: Features a calculator for quick transaction entry and a history list
- **Navigation Menu**: Access all features from the sidebar or app bar
- **Search**: Use the search bar to quickly find accounts, categories, payees, or transactions
- **Workspace Selector**: Switch between different workspaces (if you're part of multiple)

## Calculator - Quick Transaction Entry

The calculator is the fastest way to add transactions to your accounts.

### How to Use

1. **Enter Amount**: Use the number pad to enter the transaction amount
2. **Select Account**: Choose which account this transaction belongs to
3. **Select Category**: Choose an income or expense category
4. **Select Payee** (optional): Choose who you're transacting with
5. **Add Note** (optional): Add any additional notes about the transaction
6. **Save**: Tap the save button or press Enter to create the transaction

### Calculator Features

- **History List**: See your recent transactions below the calculator
- **Top Used Values**: Frequently used amounts appear as quick-select buttons
- **Operations**: Use +, -, √ó, √∑ for calculations before saving
- **Keyboard Support**: Use your keyboard for faster entry
- **Auto-scroll**: History automatically scrolls to show the latest transaction

### Tips

- üí° **Quick Entry**: The calculator remembers your last selections, making repeated entries faster
- üí° **Keyboard Shortcuts**: Press `Enter` to save, `Backspace` to delete digits
- üí° **Calculations**: Enter `100 + 50` to quickly calculate totals before saving
- üí° **Top Values**: Frequently used amounts (like $20, $50, $100) appear as buttons for one-click entry

## Managing Accounts

Accounts represent your financial accounts (checking, savings, credit cards, etc.).

### Adding an Account

1. Navigate to **Accounts** from the main menu
2. Tap the **+** button or "Add Account" button
3. Fill in the form:
   - **Name**: e.g., "Chase Checking", "Savings Account"
   - **Account Type**: Choose from Checking, Savings, Credit Card, Investment, etc.
   - **Initial Balance**: Set your starting balance (default: 0)
4. Tap **Save**

### Editing an Account

1. Go to **Accounts** page
2. Tap on the account you want to edit
3. Tap the **Edit** button (pencil icon) in the top bar
4. Modify the account details
5. Tap **Save**

### Deleting an Account

1. Open the account details page
2. Tap the **Delete** button (trash icon) in the top bar
3. Confirm the deletion

‚ö†Ô∏è **Warning**: Deleting an account will also delete all associated transactions. Make sure to export your data first if needed.

### Viewing Account Details

- **Balance**: Current account balance is displayed at the top
- **Transactions**: View all transactions for this account
- **Charts**: Visual representation of account activity
- **Filter & Sort**: Use filters to find specific transactions

### Tips

- üí° **Multiple Accounts**: Create separate accounts for different purposes (e.g., "Personal Checking", "Business Account")
- üí° **Account Types**: Use appropriate account types for better organization and reporting
- üí° **Initial Balance**: Set your current balance when creating an account to start tracking from today

## Managing Budgets

### Editing a Budget

2. Tap on the budget you want to edit

## Managing Transactions

Transactions are the core of your financial tracking.

### Adding a Transaction

**Method 1: Using the Calculator (Recommended)**

- Use the calculator on the home page for quick entry (see [Calculator section](#calculator---quick-transaction-entry))

### Editing a Transaction

1. Navigate to the transaction (via account, category, payee, or transaction list)

3. Tap the **Edit** button

5. Tap **Save**

1. Open the transaction details
2. Tap the **Delete** button
3. Confirm the deletion

‚ö†Ô∏è **Warning**: Deleting a transaction will update the account balance. Make sure this is what you want.

- **Filter**: Filter by account, category, payee, date range, or workspace members
- **Sort**: Sort by date, amount, or other fields
- **Pagination**: Navigate through large transaction lists efficiently

### Tips

- üí° **Regular Entry**: Enter transactions daily or weekly to stay on top of your finances
- üí° **Detailed Notes**: Add notes to transactions for better tracking (e.g., "Birthday gift for Mom")
- üí° **Use Calculator**: The calculator is the fastest way to add transactions

## Importing Data

Import transactions from PDF statements or CSV files to save time.

### Importing from PDF (Credit Card Statements)

1. Navigate to **Import** from the main menu
2. Select your PDF file (credit card statement)
3. Choose the date format used in your statement (DD/MM/YYYY, MM/DD/YYYY, etc.)
4. Tap **Upload**
5. Review automatically matched transactions
6. **Map Unmapped Transactions**:
   - For each unique transaction description, assign:
     - **Account**: Which account this transaction belongs to
     - **Category**: Income or expense category
     - **Payee**: Who you transacted with
7. Tap **Save All** to import all mapped transactions
8. Or tap **Ignore** to skip unmapped transactions

### Importing from CSV

1. Go to **Settings** ‚Üí **Data Management**
2. Select **Import CSV**
3. Choose your CSV file
4. Map CSV columns to My Money fields
5. Review and confirm import

### Import Tips

- üí° **Date Format**: Make sure to select the correct date format for your PDF statements
- üí° **Auto-Matching**: The system automatically matches transactions that look similar to existing ones
- üí° **Bulk Mapping**: Map all transactions with the same description at once
- üí° **Card Number**: If importing credit card statements, set the card number and account mapping for automatic assignment
- üí° **Review Before Saving**: Always review imported transactions before saving to ensure accuracy

## Workspaces & Collaboration

Workspaces allow you to collaborate with family members, roommates, or team members on shared finances.

### Creating a Workspace

1. Navigate to **Workspaces** from the main menu
2. Tap **Create Workspace**
3. Enter a workspace name (e.g., "Family Budget", "Roommate Transactions")
4. Tap **Create**

You automatically become the **Owner** of the workspace you create.

### Understanding Roles

- **Owner**: Full control, can manage members, delete workspace
- **Admin**: Can invite members, manage settings, edit all data
- **Member**: Can view and edit transactions, accounts, categories, etc., but cannot manage members

### Inviting Users to a Workspace

1. Go to **Workspaces** ‚Üí Select your workspace ‚Üí **Settings**
2. Tap **Invite Member** button
3. Enter the user's email address
4. Select their role (Member, Admin, or Owner)
5. Tap **Send Invitation**

The user will receive an email with an invitation link. They can accept the invitation by clicking the link or from the **Workspaces** page when logged in (see Accepting an Invitation).

### Accepting an Invitation

**From the email link:**

1. Check your email for the invitation
2. Click the invitation link
3. You'll be redirected to the application
4. The workspace will be added to your workspace list

**From the Workspaces page:**

1. Go to **Workspaces** in the app
2. If you have pending invitations, a **"Pending invitations for you"** section appears above your workspace list
3. Click an invitation to accept it and join the workspace
4. The workspace is added to your list; you can open its settings or load it from there

### Managing Workspace Members

1. Go to **Workspaces** ‚Üí Select workspace ‚Üí **Settings**
2. **Change Role**: Use the role dropdown next to a member's name
3. **Remove Member**: Tap the remove icon next to a member's name
4. **Cancel Invitation**: Cancel pending invitations if needed

### Workspace Features

- **Shared Data**: All accounts, transactions, categories, and budgets are shared within the workspace
- **Real-Time Updates**: See changes made by other members in real-time
- **Member Filtering**: Filter reports and transactions by workspace member
- **Multiple Workspaces**: You can be a member of multiple workspaces

### Tips

- üí° **Clear Naming**: Use descriptive workspace names (e.g., "2024 Family Budget", "Apartment Transactions")
- üí° **Role Management**: Only give Admin or Owner roles to trusted members
- üí° **Regular Communication**: Coordinate with workspace members to avoid duplicate entries
- üí° **Separate Workspaces**: Create separate workspaces for different purposes (personal, business, shared transactions)

## Version History

Version history shows a timeline of changes for an entity (account, category, payee, budget, or transaction). Use it to see who changed what and when, and to understand changes when resolving conflicts.

### Viewing Version History

1. Open the **detail page** for the entity (Account, Category, Payee, Budget, or Transaction)
2. Scroll to the **Version History** section at the bottom
3. The timeline shows each version with:
   - **Version number** and a "Current" badge for the latest
   - **Date and time** of the change
   - **Editor** (user email who made the change)
   - **Previous values** for that version (e.g. name, balance, amount)

### When Version History Helps

- **After a conflict**: When the conflict resolution dialog appears, open the entity's detail page and check Version History to see what changed before choosing a version or merging
- **Auditing**: See who updated an account, category, payee, budget, or transaction and when
- **Collaboration**: In shared workspaces, review changes made by other members

### Tips

- üí° **Detail page required**: Version history is only available on entity detail pages, not in list views
- üí° **Limit**: The panel shows up to 50 versions by default (newest first)
- üí° **Entity types**: Available for Accounts, Categories, Payees, Budgets, and Transactions

- Select your preferred currency (USD, EUR, GBP, etc.)

**Export Data**

- Export all your data (accounts, transactions, budgets, etc.) as a backup
- Useful before making major changes or for migration

- Import data from CSV files
- See [Importing Data](#importing-data) section for details

**Reset Data** ‚ö†Ô∏è

- Delete all your data and start fresh

### Tips

- üí° **Backup Regularly**: Export your data periodically as a backup
- üí° **Currency Consistency**: Make sure all workspace members use the same currency
- üí° **Date Format**: Choose a format that matches your local conventions

## Tips & Tricks

### General Tips

1. **Start Simple**: Begin with a few accounts and categories, then expand as needed
2. **Be Consistent**: Use consistent naming for accounts, categories, and payees
3. **Regular Updates**: Enter transactions regularly (daily or weekly) to stay on top of your finances
4. **Use Notes**: Add notes to transactions for better tracking and future reference
5. **Review Reports**: Check your reports monthly to understand spending patterns

### Productivity Tips

1. **Calculator Shortcuts**: Use the calculator for quick entry instead of manual forms
2. **Top Used Values**: Frequently used amounts appear as quick-select buttons
3. **Search Everything**: Use the search bar to quickly find any account, category, payee, or transaction
4. **Keyboard Navigation**: Use keyboard shortcuts for faster navigation
5. **Bulk Import**: Import credit card statements instead of manual entry

### Organization Tips

1. **Account Structure**: Create separate accounts for different purposes (Personal, Business, Savings)
2. **Category Hierarchy**: Use specific category names (e.g., "Restaurants" vs "Fast Food")
3. **Payee Consistency**: Always use the same name for the same payee
4. **Budget Strategy**: Start with budgets for your biggest expense categories
5. **Workspace Organization**: Use separate workspaces for different financial contexts

### Collaboration Tips

1. **Clear Communication**: Coordinate with workspace members to avoid conflicts
2. **Role Assignment**: Assign appropriate roles based on trust and responsibility
3. **Regular Sync**: Check in regularly to ensure everyone is entering transactions
4. **Shared Categories**: Agree on category names and usage with your team
5. **Member Filtering**: Use member filters in reports to see individual contributions

### Troubleshooting Tips

1. **Balance Mismatch**: If account balance seems wrong, check for duplicate or missing transactions
2. **Import Issues**: Verify date format matches your statement format
3. **Search Not Working**: Try different search terms or check spelling
4. **Missing Data**: Ensure you're viewing the correct workspace
5. **Performance**: If the app feels slow, try filtering or paginating large transaction lists
